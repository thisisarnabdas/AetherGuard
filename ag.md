## üöÄ Quick Start

1. Clone the repository:
   ```
   git clone https://github.com/thisisarnabdas/AetherGuard.git
   ```
2. Navigate to project directory:
   ```
   cd AetherGuard
   ```
3. Install dependencies
   ```
   pip install -r requirements.txt
   ```

4. Run the application
   ```
   python run.py
   ```
